## 📖 Deskripsi Proyek
Repository ini berisi kumpulan tugas praktikum mata kuliah **Algoritma dan Struktur Data**. Fokus utama dari proyek ini adalah mengimplementasikan logika pemrograman menggunakan bahasa **Python** untuk menyelesaikan berbagai masalah komputasi, mulai dari logika dasar hingga manipulasi struktur data kompleks.

Setiap modul dirancang untuk memperkuat pemahaman mengenai efisiensi kode, alur kontrol, dan manajemen data dalam lingkungan pengembangan berbasis notebook (Google Colab).

Modul ini mengeksplorasi kemampuan dasar kontrol alur program dan pemecahan masalah matematis.
* **Soal 1: Algoritma FizzBuzz** Implementasi logika pembagian menggunakan operator modulus. Program akan mengevaluasi angka berdasarkan kondisi tertentu (kelipatan 3, 5, atau keduanya) untuk melatih pemahaman struktur `if-elif-else`.
* **Soal 2: Sistem Konversi Waktu** Program yang mengubah input jumlah hari mentah menjadi format tahun, bulan, dan sisa hari. Fokus utama adalah pada operasi pembagian bulat (`//`) dan sisa bagi (`%`).
